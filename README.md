
Description


Diff Agent helps developers turn code changes into safer, cleaner software faster. The user pastes a GitHub repository link and a git diff, and the app analyzes the change using IBM watsonx Granite through specialized AI agents. One agent reviews the diff for bugs, edge cases, and repo-aligned improvements, while another provides alternative implementation ideas and tradeoffs.

The project was built for the IBM Bob Hackathon with IBM Bob used as a development partner throughout the process. Bob helped generate the initial app structure, debug Python and watsonx API issues, improve the user interface, and refine the project into a working MVP. The goal is to make code review more accessible for beginner developers, hackathon teams, and small software teams by giving them fast, context-aware feedback before code reaches production.

Diff Agent directly supports the challenge theme, “Turn idea into impact faster,” by reducing the time developers spend manually reviewing changes, explaining risks clearly, and helping builders ship with more confidence.
